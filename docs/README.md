# usage
```
$ cd ~
$ npx git clone git@github.com:swappet/acu.fund-books.git
$ cd acu.fund-books
$ npx npm install
$ npx npm run docs:dev
```

打开 http://localhost:8080/ 即可看到一个 Vue 文档风格的网站
