# acu.fund-books
blockchain and open source book of ACU.Fund with VuePress and github.io base on WTFPL.

# usage
```
$ cd ~
$ npx git clone git@github.com:swappet/acu.fund-books.git
$ cd acu.fund-books
$ npx npm install
$ npx npm run docs:dev
```
